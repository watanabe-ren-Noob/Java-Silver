第1章 Javaの概要とプログラム作成

1. ソースコードの構成ルール

順序：package → import → class

  • java.langパッケージは自動インポートされる。
  • 同一パッケージ内のクラスもインポート不要。

2. mainメソッドとstaticの制約

staticメソッドから非static（インスタンス）変数へは直接アクセス不可。

3. 実行コマンドと引数

  • 実行：java Sample (.classは不要)
  • 引数：args[0] はクラス名の次の単語から。

第2章 基本データ型と文字列の操作

1. 数値リテラルと識別子

  • アンダースコア(_)：数字の間のみOK。先頭、末尾、小数点前後はNG。
  • 8進数：0から始まる数値。8と9は使えない（例：0827はエラー）。
  • 識別子：使用可能な記号は $ と _ のみ。

2. ローカル変数型推論 (var)

メソッド内のローカル変数でのみ使用可能。

  • NG：フィールド（クラス変数）、引数、戻り値の型。
  • NG：初期化なし、nullでの初期化。

3. String と StringBuilder

String（不変）：一度作成すると内容は変わらない。replace等は「新しい文字列」を返すだけ。

StringBuilder（可変）：appendやreverseで自身の内容を書き換え可能。

テキストブロック：""" で開始。開始直後の改行は必須。インデントは終端引用符の位置が基準。

4. 配列の宣言と初期化

  • 宣言：int[] a はOKだが、int[2] a のようにサイズ指定は不可。
  • 初期化：new int[2]{1, 2} はNG（サイズと要素の同時指定不可）。
  • 多次元：最初の次元のサイズは省略不可。new int[][3] はNG。
